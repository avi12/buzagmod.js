### Getting started
1. Clone the repository
2. Install [PNPM](https://pnpm.io/installation)
3. ```shell
   pnpm i
   pnpm dev
   ```

### To build the app:
```shell
pnpm build
   ```

### To build the executable:
```shell
pnpm release
```
